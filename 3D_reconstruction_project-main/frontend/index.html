About MedView 3D

MedView 3D is a web-based 3D reconstruction system for medical images (CT/MRI) with interactive visualization.

This application allows medical professionals to:

  • Load and process DICOM medical images
  • Segment structures of interest
  • Generate 3D reconstructions
  • Interactively explore 3D models alongside original slice data

The processing pipeline includes:

  1. DICOM Loading: Import CT/MRI data
  2. Preprocessing: Convert to HU units and resample
  3. Segmentation: Identify structures of interest
  4. 3D Reconstruction: Create interactive 3D model
